## 📘 Descripción general del sistema
Este proyecto corresponde a la actividad de la Semana 4 de POO. Se trata de una aplicación Java que lee datos de tours turísticos desde un archivo de texto externo, los carga en una colección `ArrayList` y permite recorrer y filtrar los registros por consola. El caso de uso corresponde a la agencia de turismo **Llanquihue Tour**.

## 🧱 Estructura general del proyecto

```plaintext
📁 src/
├── model/      # Clase Tour
├── data/       # Clase GestorDatos
└── ui/         # Clase principal Main
📁 resources/
└── tours.txt
└── README.md
```

## ⚙️ Instrucciones para clonar y ejecutar el proyecto

1. Clona el repositorio desde GitHub:

```bash
git clone https://github.com/nicosalgadogit/llanquihue-tour
```

2. Abre el proyecto en IntelliJ IDEA.

3. Verifica que el archivo `tours.txt` esté correctamente ubicado en `resources/`.

4. Ejecuta el archivo `Main.java` desde el paquete `ui`.
